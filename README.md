This project demonstrates how to work with routing in React. 
User can switch theme by pressing button.

Technologies 

  React
  TypeScript
  RouterProvider
  createBrowserRouter
  Outlet
  NavLink

Install project

  clone project:
    git@github.com:yaroslavnakonechniy/home-work-43.git

  Install dependencies:

    npm install
    npm install react-router-dom
    npm install react-router

  Run dev server:

    npm run dev

  Open in browser:

    http://localhost:5173`
